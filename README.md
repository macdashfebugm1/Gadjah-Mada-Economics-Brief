Gadjah Mada Economics Brief: Macroeconomics

External consistency check between Indonesia's official GDP components (household consumption and gross fixed capital formation) and independent activity indicators, with a break at 2025 Q2.

Files
hh_gfcf_external_consistency.R: main analysis script.
gdp_all.xlsx: quarterly GDP and components.
indicators.xlsx, indicators2_no_electricity.xlsx: monthly and quarterly activity indicators.

How to run
Open the R script, set the working directory to the repository folder, and run end to end. Required packages are installed automatically on first run. Outputs are written to subfolders prefixed outputs_, tables_, and figures_.
Data retrieved via CEIC; original sources include BPS, Bank Indonesia, OJK, Gaikindo, S&P Global, etc.
